# Nickel Alkoxide IRC Analysis

This repository provides scripts and notebooks to analyze **Gaussian** and **ORCA** IRC outputs for nickel-alkoxide complexes.

## Folder Structure
- `data/` – Raw IRC log/output files
- `scripts/` – Python scripts for dipole and Hirshfeld spin extraction
- `notebooks/` – Jupyter notebooks for analysis and plotting
- `results/` – Generated CSV/Excel files

## Usage

```bash
# Run all analyses
python scripts/run_analysis.py
